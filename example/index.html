@marianmeres/vanilla

  • Fetch

    Async request flow — observable state, computed loading flag, per-state <template> (idle / loading / error / result), and request cancellation via AbortController.

  • Todo

    Classic list app — microtask-batched updates, reactTo, computed values, and a global light/dark theme toggle.

  • Todo (SSR)

    Progressive enhancement — server-rendered markup brought to life in place with enhance, no client-side rebuild of the list.

  • Multi-component

    Single-file components loaded with loadComponent() — props, scoped markup/script/style per file, wired up through an import map. Requires deno task example:build first.

  • Quake Watch

    Live USGS earthquake feed — fetch, client-side magnitude filtering, and severity colors driven entirely by @marianmeres/design-tokens, with swappable themes and persisted light/dark preference.
